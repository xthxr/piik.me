# Contributing to piik.me

Thank you for your interest in contributing to piik.me! We welcome contributions from the community to help improve this project.

1.  **Fork and Clone**
    ```bash
    git clone https://github.com/xthxr/piik.me.git
    cd piik.me
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Configuration**
    -   Copy `.env.example` to `.env` and configure your environment variables.
    -   Set up your Firebase credentials in `public/js/firebase-config.js` (copy from example).

4.  **Run Locally**
    ```bash
    npm start
    ```
    Visit `http://localhost:3000`.
